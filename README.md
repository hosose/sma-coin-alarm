# 📈 Crypto Trend Signal Bot (Weekly SMA)

이 프로젝트는 **비트코인(BTC)**과 **이더리움(ETH)**의 장기 추세를 추적하여 매수/매도 타이밍을 알려주는 텔레그램 봇입니다.

별도의 서버 비용 없이 **GitHub Actions**를 이용하여 매일 자동으로 시세를 감시합니다.

## 🚀 주요 기능

* **자동 실행:** 매일 한국 시간 오전 9:30에 자동으로 실행되어 시장을 분석합니다.
* **주봉 분석:** 일봉의 노이즈를 제거하고 큰 추세(Big Wave)를 잡기 위해 **주봉(Weekly)** 데이터를 사용합니다.
* **스마트 알림:**
  * 매수/매도 신호 발생 시: **즉시 알림**
  * 매주 월요일: 신호가 없어도 현재 추세 **정기 브리핑**
  * 그 외 요일: 특이사항 없으면 **알림 없음** (조용한 봇)
* **비용 0원:** GitHub Actions의 무료 티어를 사용하므로 서버 유지비가 들지 않습니다.

## 📊 매매 전략 (Strategy)

이 봇은 **이동평균선 교차(Moving Average Crossover)** 전략을 사용합니다.

* **타임프레임:** 주봉 (1 Week)
* **지표:**
  * **SMA 5 (5주 이동평균선):** 단기 추세 (약 1달)
  * **SMA 20 (20주 이동평균선):** 장기 추세 (약 5달)
* **매수 신호 (Golden Cross):** 5주선이 20주선을 **상향 돌파**할 때 (🔥 진입)
* **매도 신호 (Dead Cross):** 5주선이 20주선을 **하향 이탈**할 때 (💧 청산)

> 이 전략은 횡보장에서는 잦은 매매가 발생할 수 있으나, **강한 상승장(Bull Market)과 하락장(Bear Market)**을 확실하게 발라먹는 데 특화되어 있습니다.
